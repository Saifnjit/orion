How it works

Set it up once.
Never think about it again.

The bot runs every day at 8am, 1pm, and 6pm — fully automatic.

Step 01
📡

Finds trending content

Monitors Reddit, Hacker News, TechCrunch, The Verge, Wired, and more for the most engaging AI and tech stories.

Step 02
✍️

Writes the caption

Claude AI writes a punchy, professional LinkedIn post — sharp tone, specific facts, ends with something people want to respond to.

Step 03
📊

Scores for virality

Every post is scored 1–10 before it goes live. Anything below 7 gets skipped — only the best makes it through.

Step 04
🖼️

Finds a relevant image

Claude Vision picks the most specific, relevant image — actual photos of people and companies, not random stock photos.

Step 05
🚀

Posts to LinkedIn

Automatically uploaded and published via the LinkedIn API. No browser, no manual steps. Just posts.

Features

Everything you need.
Nothing you don't.

Built lean — every part of the pipeline does exactly one job.

🔍

Multi-source content pipeline

Pulls from Reddit's top AI/tech subreddits, Hacker News top stories, and 8 professional RSS feeds simultaneously.

🤖

Claude-powered writing

Not generic AI slop. Captions are grounded in real facts, written in a natural tone that sounds like a sharp professional — not a LinkedIn guru.

🎯

Viral score filter

Every caption is scored across hook strength, emotional trigger, comment potential, and specificity. Below 7/10 doesn't post.

🖼️

Smart image selection

Wikipedia headshots, article og:images, and Bing search — Claude Vision picks the winner and rejects anything generic.

🔄

Duplicate prevention

Post history tracked for 7 days. Fuzzy title matching catches near-duplicates so you never post the same story twice.

⚡

Fully automated schedule

Posts at 8am, 1pm, and 6pm every day. Runs in the background — just leave it on and forget it.

Get started

Running in under 5 minutes.

You need a LinkedIn access token and an Anthropic API key. That's it. No database, no server, no credit card.

1
Clone the repo One command to get the code locally
2
Add your API keys to .env LinkedIn access token + Anthropic API key
3
Run python main.py Posts immediately, then 3x/day forever
# 1. Clone the repo
git clone github.com/Saifnjit/autopost
cd autopost

# 2. Install dependencies
pip install -r requirements.txt

# 3. Add your keys to .env
LINKEDIN_ACCESS_TOKEN=your_token
ANTHROPIC_API_KEY=your_key

# 4. Run it
python main.py

# That's it. Posts every day automatically.
