How It Works

Three steps. Zero effort.

The entire pipeline runs in the background while you focus on real work.

1

Fetch trending content

Scans RSS feeds from TechCrunch, Wired, The Verge, and more — plus Reddit and Hacker News — to find what's actually trending in your niche right now.

2

Generate & score the post

Claude AI writes a hook-first LinkedIn post grounded in real facts. A viral scoring model checks it — if it doesn't hit the bar, it tries again with a fresh article.

3

Publish with image

Finds a relevant image from Wikipedia, the article's og:image, or Bing. Validates it with Claude Vision so you never get a random screenshot. Then posts to LinkedIn.

Features

Everything built in

🔍

Multi-source content fetch

RSS feeds, Reddit top posts, and Hacker News — filtered by your niche keywords, with duplicates removed.

🤖

Claude AI writing

Uses Claude Sonnet to write posts in your personal voice. Configurable style — casual, professional, founder-mode, whatever fits you.

📈

Viral scoring

Each post is scored on hook strength, emotion, specificity, and call-to-action before it gets published. Low-scoring posts are skipped automatically.

📷

Smart image matching

Extracts the subject from each headline, searches Wikipedia and Bing for a real photo, then validates relevance with Claude Vision.

⚙️

Fully configurable

One settings.py file controls your niche, posting style, post times, subreddits, keywords, and viral score threshold.

🖥️

Desktop app (GUI)

Don't want to touch the terminal? Run the desktop app. Configure everything in a clean UI and start the bot with one click.

Setup

Running in 3 minutes

Clone, configure, run. That's it.

# 1. Clone the repo git clone https://github.com/Saifnjit/autopost cd autopost
# 2. Install dependencies pip install -r requirements.txt
# 3. Add your keys (or use the setup wizard) cp .env.example .env # Edit .env with your LinkedIn token + Anthropic key
# 4. Run python main.py
Need keys?

LinkedIn Access Token

Create a LinkedIn app at linkedin.com/developers and generate an OAuth access token with w_member_social scope.

Need keys?

Anthropic API Key

Sign up at console.anthropic.com and generate an API key. Claude Haiku + Sonnet are used — very cost-efficient.
